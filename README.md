 LearnHub Booking System

 Project Overview

LearnHub is a web-based educational platform designed to help users explore subjects, select courses, and manage their learning process efficiently. The system allows users to register, browse available subjects, add courses to a cart, and proceed with booking and payment. It provides a structured and user-friendly environment for students to access learning resources and complete transactions smoothly.

Purpose of the Project

The main purpose of this project is to build an interactive learning platform where users can:

 Create an account and log in
 Explore different subjects and courses
 Add selected courses to a cart
 Book courses and complete payment
 Contact support for queries

This project also demonstrates basic concepts of web development and application testing.


 Features

 User registration through Signup page
 Explore subjects and courses
 Add courses to cart
 Course booking functionality
 Payment process integration
 Contact form for user communication
 Simple and responsive user interface


Modules

 1. Signup Module

Allows users to create an account by entering their name, email, and password.

 2. Subjects Module

Displays a list of subjects such as Mathematics, Physics, Chemistry, Biology, and more. Users can explore different learning areas and available courses.

 3. Cart Module

Enables users to add selected courses to a cart. Users can view, update, or remove items before proceeding to booking.

 4. Booking Module

Allows users to confirm their selected courses from the cart and proceed with the booking process.

 5. Payment Module

Handles the payment process for booked courses. It ensures that users can complete transactions securely and successfully.

 6. Contact Module

Provides a form where users can enter their details and send messages for support or inquiries.



 Testing

The project includes testing to ensure that:

All pages load correctly
Input fields accept valid data
Invalid inputs are handled properly
Cart, booking, and payment functionalities work correctly
Navigation and user interactions perform as expected



Conclusion

The LearnHub Booking System is a complete learning platform that combines subject exploration with booking and payment features. It helps users easily select and enroll in courses while ensuring a smooth user experience. This project also enhances understanding of web application development and testing concepts.
